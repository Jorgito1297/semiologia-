# PROMPT MAESTRO — GENERADOR DE UI MÉDICA PREMIUM PARA ANTIGRAVITY

## OBJETIVO
Diseñar y renderizar interfaces médicas premium de simulación clínica con estética hospitalaria futurista, orientadas a:
* semiología médica
* auscultación cardíaca
* auscultación pulmonar
* simulación EMS
* laboratorios clínicos interactivos
* plataformas universitarias médicas

La interfaz debe parecer un sistema médico real de entrenamiento clínico de alto nivel, similar a:
* simuladores hospitalarios
* software de radiología avanzada
* monitores ICU modernos
* plataformas anatómicas interactivas
* software biomédico militar/aeroespacial

# ANATOMÍA

## Render anatómico
El torso debe:
* verse anatómicamente correcto
* incluir costillas translúcidas
* corazón iluminado
* pulmones semitransparentes
